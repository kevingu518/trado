# 文檔目錄

歡迎使用 Trado React 交易紀錄管理系統的文檔。本文檔目錄提供了所有相關文檔的索引和說明。

## 文檔列表

### 1. [系統架構文檔](ARCHITECTURE.md)

描述系統的整體架構設計，包括：
- 架構圖與模組說明
- 認證模組設計
- 交易紀錄模組設計
- 路由管理
- API 通訊層
- 狀態管理
- UI 組件架構
- 樣式管理
- 開發規範

**適合閱讀對象**：開發人員、系統架構師

---

### 2. [功能規格文檔](FEATURES.md)

詳細說明系統的所有功能規格，包括：
- 登入系統功能
  - 使用者登入
  - 使用者註冊
  - 忘記密碼
  - 路由保護
- 交易紀錄功能
  - 交易記錄列表
  - 開單（建立 Position）
  - 建倉位（新增 Fill）
  - 查看交易詳情
  - 過濾與搜尋
  - 分頁功能
- 交易檢討功能
- 未來功能規劃

**適合閱讀對象**：產品經理、開發人員、測試人員

---

### 3. [資料模型文檔](DATA_MODEL.md)

描述系統中使用的資料結構和資料模型，包括：
- 核心資料模型
  - Position（交易記錄）
  - Fill（倉位記錄）
  - Review（交易檢討）
  - User（使用者）
- 資料關係圖
- 資料驗證規則
- 計算欄位說明
- 資料初始化
- 資料持久化

**適合閱讀對象**：開發人員、後端工程師、資料庫設計師

---

### 4. [API 規格文檔](API.md)

描述系統的 API 介面規格，包括：
- API 基礎配置
- 認證相關 API
  - 使用者登入
  - 使用者註冊
  - 忘記密碼
  - 登出
  - 刷新 Token
- 交易相關 API
  - 取得交易記錄列表
  - 取得單筆交易記錄
  - 建立交易記錄
  - 更新交易記錄
  - 刪除交易記錄
  - 新增/更新/刪除倉位記錄
  - 更新交易檢討
- 使用者相關 API
- 錯誤處理
- API 調用範例
- 目前實作狀態

**適合閱讀對象**：前端開發人員、後端開發人員、API 測試人員
